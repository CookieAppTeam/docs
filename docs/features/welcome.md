# Welcome Messages
Greet new members with **personalized welcome messages** when they join your server.

- Use `/welcome` to configure welcome messages.

!!! tip "Customize Welcome Messages"
    You can customize welcome messages using the `/messages` command.

## Customization
The **design** of the welcome message can be fully customized using the `/messages` command.

Additionally, you can add different **texts** that will be selected at random when a 
new member joins. These texts will be inserted into your custom message design.
