# Cookie Documentation
This repository contains the documentation for [Cookie](https://cookieapp.me), a multifunctional Discord bot.

The documentation is built using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

## Contributing
You can run the documentation locally to contribute:
1. Install the most recent version of [Python](https://www.python.org/downloads/).
2. Clone this repository.
3. Install the required dependencies:
   ```bash
   pip install -r docs/requirements.txt
   ```
4. Preview the documentation:
   ```bash
    mkdocs serve --livereload
    ```
5. Make your changes and submit a pull request.
