# Economy
Cookie provides an economy system that allows users to collect cookies and spend them on various 
items.

!!! info
    The economy system is global. Cookies and items are shared across all servers where you use 
    Cookie.

!!! tip "Daily Reminder"

Grow your own cookie ingredients using the `/farm` command.

### What are the benefits of farm upgrades?
- You can buy **more fields** to grow ingredients.
- You unlock **new plants** that you can cultivate.
- You get **more harvest** per field.

### What are server investments?
- You can invest cookies in the current server to unlock benefits for **all members**.
- For each level achieved, you can grow one additional plant **for free** forever.
