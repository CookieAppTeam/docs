# Statistics & Leaderboards
Cookie provides detailed statistics and leaderboards for **text** and **voice** activity.

- Use `/stats` to view activity stats for the **server** or **individual members**.
- View the **server leaderboard** with `/leaderboard server`.

## Server Info
Use `/info server` to view the history of the server's member count and boosts.

## Server Boosters
Use `/booster` to view a leaderboard of top server boosters.

- You can choose to display current boosters or all-time boosters.
