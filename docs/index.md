## Invite Cookie { #invite }
Visit [Cookie's Website](https://cookieapp.me/invite) to add Cookie to your Discord server or
Discord account.

- On servers, you can use all of Cookie's features.
- If you add Cookie to your Discord account, you can use some of Cookie's features, like games, 
in private messages.

## Configure Cookie for your server
- Use `/settings` to adjust Cookie's settings for your server.
- Use `/help` to see a list of all available commands and features.


Cookie also has a [Dashboard](https://cookieapp.me), but you can still configure everything within 
Discord.
